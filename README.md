# 🖥️ Dashboard

The web application allows users to predict molecular properties from a SMILES string, visualize the molecular structure, and evaluate drug-likeness in real time.

### Prediction Interface

<p align="center">
  <img src="images/dashboard_prediction.png" width="900"/>
</p>

Users can:

- Enter a SMILES molecule.
- Generate Morgan fingerprints.
- Predict **logP**, **QED**, and **SAS**.
- Evaluate Lipinski's Rule of Five.
- Visualize the molecular structure instantly.
